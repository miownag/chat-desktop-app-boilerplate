# Chat Desktop App Boilerplate

A modern **Full-stack AI chatbot desktop application** built with TypeScript, React, Tauri and Hono. This boilerplate provides a solid foundation for building your own AI-powered desktop applications.

### Prerequisites

First, ensure you have **[Bun](https://bun.sh/docs/installation)** installed on your system.

### Installation & Development

```bash
# Install dependencies
bun install

# Start the development server
bun dev
```

That's it! The application will automatically open in development mode.
